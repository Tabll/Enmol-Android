# Enmol
以沫-恋人间的交互应用

## 前言
此项目试图开发出一个面向恋人们的应用程序，现阶段以 **网页**、**Android**、**UWP** 客户端的开发为主

## 功能目标
- [ ] P2P即时聊天
- [ ] 日历日程表共享
- [ ] 共享/私有备忘录
- [ ] 文件传输
- [ ] 远距离大致方位显示
- [ ] 事件记录
- [ ] 实时交互小游戏

## 现状
- v0.0.1正在开发中
  - 以数据库的构建为主
